# Icon Placeholders

This directory should contain the following icon files:

- icon16.png (16x16) - Default extension icon
- icon32.png (32x32) - Default extension icon  
- icon48.png (48x48) - Default extension icon
- icon128.png (128x128) - Default extension icon
- icon-active-32.png (32x32) - Active state when policies detected

You can generate these icons using:
- A design tool like Figma, Sketch, or Canva
- Online icon generators
- Or use existing privacy/security themed icons

Suggested design:
- Shield or lock icon with a magnifying glass
- Blue color scheme (#3B82F6)
- Clean, modern style
